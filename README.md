Documentation: -

⦁	Header → Navigation & Branding
The header contains the logo and menu links, helping users identify the institute and navigate easily through the website.

⦁	Hero → First Impression & Call-to-Action
The hero section is the first visible part of the website. It attracts attention and encourages users to take action like enrolling or downloading a brochure.

⦁	Sidebar → Quick Navigation
The sidebar provides quick links to important courses, allowing users to access content faster without searching.

⦁	Course Box → Course Presentation
Course boxes display individual course details such as name, description, and action buttons in an organized and attractive card format.

⦁	Features → Trust Building
The features section highlights key benefits like expert faculty and placement support, which builds trust and confidence among users.

⦁	About → Institute Information
The about section explains why the institute is valuable and provides detailed information about training quality and services.

⦁	Footer → Copyright & Closure
The footer appears at the bottom of the website and provides copyright information, giving the site a professional ending.

⦁	Offcanvas → Responsive Navigation
Offcanvas menus are used mainly on mobile devices to show navigation links without occupying screen space.

⦁	Carousel → Visual Engagement
The carousel displays multiple images in a sliding format, making the website more attractive and engaging for users.
